# RoxWealth


# Clone the repo

```bash
git clone https://github.com/Rohit-118/RoxWealth.git
```
### npm version "10.2.3" Required
# Navigate into the my-json-server folder

```bash
cd RoxWealth/my-json-server
```


## Setup

### Install dependencies

```bash
npm install
```

### Run the app in development mode

```bash
npm start
```
